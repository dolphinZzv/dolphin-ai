# Data Analysis Skill

When performing data analysis:

1. Understand the data structure and schema
2. Clean and validate the data before analysis
3. Use appropriate statistical methods
4. Visualize results when helpful
5. Provide actionable insights with confidence levels

Output format: summary of findings, methodology used, key numbers, and recommendations.
