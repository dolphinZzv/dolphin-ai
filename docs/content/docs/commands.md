---
title: 命令
---

Dolphin 支持以下斜杠命令：

| 命令 | 描述 |
|------|------|
| `/help` | 显示帮助信息 |
| `/models list` | 列出所有可用模型 |
| `/models use <name>` | 切换到指定模型 |
| `/session new` | 创建新会话 |
| `/session list` | 列出所有会话 |
| `/session use <id>` | 切换到指定会话 |
| `/mcp` | 列出已加载的 MCP 工具 |
| `/skills list` | 列出已加载的技能 |
| `/context` | 显示系统提示词 |
| `/version` | 显示版本号 |
| `/scheduler list` | 列出计划任务 |
